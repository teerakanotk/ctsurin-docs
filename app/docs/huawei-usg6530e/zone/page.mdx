# Zone

เขตหรือกลุ่มของเครือข่าย ที่ถูกจัดกลุ่มตามระดับความเชื่อถือ (trust level) หรือลักษณะการใช้งาน เพื่อใช้กำหนดนโยบายความปลอดภัยในการควบคุมการรับ-ส่งข้อมูลระหว่าง Zone ต่าง ๆ ได้อย่างมีประสิทธิภาพและปลอดภัย

## Default Zone

ระบบ Firewall มักกำหนด Zone เริ่มต้น (Default Zone) เพื่อจัดลำดับความเชื่อถือของเครือข่ายแต่ละประเภท โดยแต่ละ Zone จะมีระดับความสำคัญ (Priority) ที่แตกต่างกัน ซึ่งมีผลต่อการควบคุมการไหลของทราฟฟิกระหว่าง Zone ต่าง ๆ ดังนี้:

- local (Priority: 100) แทนตัวอุปกรณ์ Firewall เอง ใช้สำหรับการจัดการหรือควบคุมอุปกรณ์ เช่น การเข้าถึงผ่าน SSH, HTTPS, หรือ SNMP

- trust (Priority: 85) เป็นโซนสำหรับเครือข่ายภายใน (Internal Network) ซึ่งถือว่าเป็นเครือข่ายที่เชื่อถือได้ เช่น LAN ขององค์กร หรือเครือข่ายสำหรับพนักงาน

- dmz (Priority: 50) ย่อมาจาก Demilitarized Zone เป็นโซนสำหรับเซิร์ฟเวอร์หรือบริการที่ต้องการให้บุคคลภายนอกเข้าถึงได้ เช่น Web Server, Mail Server หรือ DNS Server โดยออกแบบให้แยกออกจากเครือข่ายภายใน (trust) เพื่อป้องกันไม่ให้ความเสียหายจากการบุกรุก DMZ ส่งผลกระทบต่อระบบหลัก

- untrust (Priority: 5) เป็นโซนสำหรับเครือข่ายภายนอก (External Network) หรือเครือข่ายที่ไม่น่าเชื่อถือ เช่น อินเทอร์เน็ต โดยปกติ Firewall จะไม่อนุญาตให้ทราฟฟิกจากโซนนี้เข้าถึงโซนที่มีความเชื่อถือสูงกว่า เว้นแต่ว่าจะมีการกำหนดนโยบายไว้โดยเฉพาะ

## หลักการทำงานของ Zone และระดับความสำคัญ (Priority)

- เมื่อมีทราฟฟิกพยายามส่งจากโซนที่มีระดับความเชื่อถือ (Priority) สูงกว่า ไปยังโซนที่มีระดับความเชื่อถือต่ำกว่า (เช่น จาก Trust ไปยัง Untrust) ระบบ Firewall มักจะ อนุญาต ทราฟฟิกนั้นโดยอัตโนมัติ เว้นแต่ว่าจะมีนโยบาย (Policy) ที่ระบุให้ปฏิเสธ ไว้อย่างชัดเจน

- ในทางกลับกัน หากทราฟฟิกพยายามส่งจากโซนที่มีระดับความเชื่อถือต่ำกว่า ไปยังโซนที่มีระดับความเชื่อถือสูงกว่า (เช่น จาก Untrust ไปยัง Trust หรือ DMZ) ระบบ Firewall จะ ปฏิเสธ ทราฟฟิกนั้นโดยค่าเริ่มต้น เว้นแต่ว่าจะมีนโยบายที่อนุญาตไว้อย่างชัดเจน

## การจัดการ Zone

<Steps>

### เข้าสู่เมนู Network > Zone

ไปที่เมนู Network แล้วเลือก Zone ที่แท็บ Zone List จะแสดงรายการของ Zone ที่มีอยู่ในระบบ

![sec-zone-1](/images/huawei-usg6530e/sec-zone-1.png)

### เลือก Zone ที่ต้องการจัดการ

คลิกที่ Zone ที่ต้องการแก้ไขหรือปรับแต่ง เพื่อเข้าสู่หน้าการตั้งค่า

![sec-zone-2](/images/huawei-usg6530e/sec-zone-2.png)

### แก้ไขข้อมูลในหน้าต่าง Modify Zone

ในหน้านี้คณสามารถปรับแต่งข้อมูลของ Zone ได้ตามต้องการ เช่น

- เพิ่มหรือนำอินเทอร์เฟซ (Interface) เข้า/ออก จาก Zone

- แก้ไขคำอธิบาย (Description) ของ Zone

![sec-zone-3](/images/huawei-usg6530e/sec-zone-3.png)

### บันทึกการเปลี่ยนแปลง

เมื่อทำการปรับแต่งเสร็จสิ้น ให้คลิกที่ปุ่ม OK เพื่อบันทึกการเปลี่ยนแปลง

</Steps>
