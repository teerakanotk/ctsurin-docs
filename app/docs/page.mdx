# Introduction

### Server

- Nginx Proxy Manager: บอกลากับความยุ่งยากในการจัดการ Reverse Proxy และ SSL Certificate ด้วยหน้าจอ UI ที่ใช้งานง่าย

- Proxmox VE: เรียนรู้วิธีสร้างและจัดการ Virtual Machine (VM) และ Container ได้อย่างง่ายดายด้วยแพลตฟอร์ม Open-Source ที่ทรงพลัง

- Technitium DNS: จัดการ DNS ของคุณเองได้อย่างอิสระและยืดหยุ่น เพิ่มความปลอดภัยและความเร็วในการใช้งาน

- Zabbix: เฝ้าระวังระบบของคุณแบบเรียลไทม์! ตั้งแต่เซิร์ฟเวอร์, แอปพลิเคชัน, ไปจนถึงฐานข้อมูล ไม่พลาดทุกการแจ้งเตือน

### Firewall

- Huawei USG6000E: ปกป้องเครือข่ายของคุณให้ปลอดภัยจากภัยคุกคามต่าง ๆ ด้วย Firewall ประสิทธิภาพสูง

### Switch

- Huawei S5700: ตั้งค่าและจัดการเครือข่ายภายในของคุณให้มีประสิทธิภาพสูงสุดด้วย Switch ระดับ Enterprise
