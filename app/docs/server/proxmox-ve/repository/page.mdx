# Proxmox VE Repository

Repository คือแหล่งเก็บรวบรวมไฟล์ ซอฟต์แวร์ แพ็คเกจ หรือโค้ดต่างๆ ที่มีการจัดระเบียบไว้อย่างเป็นระบบ เพื่อให้ผู้ใช้งานหรือระบบสามารถเข้าถึง ดาวน์โหลด และติดตั้งได้อย่างสะดวกสบาย ลองนึกภาพว่า Repository เป็นเหมือนกับห้องสมุดขนาดใหญ่ที่เก็บ "หนังสือ" (แพ็คเกจซอฟต์แวร์) จำนวนมากไว้ แต่ละ "หนังสือ" มีชื่อเรียกเฉพาะ เวอร์ชันที่ชัดเจน และถูกจัดหมวดหมู่ไว้อย่างดี เมื่อคุณต้องการ "หนังสือ" เล่มไหน คุณก็แค่ไปที่ห้องสมุดแล้วหยิบออกมาได้เลย

## ขั้นตอนการอัปเดต Repository

กรณีไม่ได้สมัคร Enterprise-Subscription แนะนำให้ปิดใช้งาน Enterprise Repository

<Steps>

### ปิดใช้งาน Enterprise Repository

- เข้าสู่หน้า Web UI ของ Proxmox VE

```txt copy
http://<ip_address>:8006
```

- ที่แถบนำทางด้านซ้ายมือ ให้เลือก **node** ที่ต้องการอัปเดต (เช่น pve1)

- ไปที่เมนู **Updates** > **Repositories**

- เลือกบรรทัดของ **pve-enterprise** repository แล้วคลิกปุ่ม **Disable**

![proxmox-repository-1](/images/proxmox/repository-1.png)

### เพิ่ม No-Subscription Repository

- คลิก **Add** ที่อยู่บริเวณส่วนของ **APT Repositories**

![proxmox-repository-2](/images/proxmox/repository-2.png)

- ในหน้าต่างที่ปรากฏขึ้น ให้เลือก **No-Subscription** แล้วคลิกปุ่ม **Add**

![proxmox-repository-3](/images/proxmox/repository-3.png)

### อัปเดตแพ็คเกจ

- ไปที่เมนู **Updates**

- คลิกปุ่ม **Refresh**

![proxmox-repository-4](/images/proxmox/repository-4.png)

- รอจนกระทั่งกระบวนการเสร็จสมบูรณ์และแสดงข้อความ **Task OK**

![proxmox-repository-5](/images/proxmox/repository-5.png)

- กดปุ่ม **Esc** เพื่อปิดหน้าต่างแสดงผล

### อัปเกรดแพ็คเกจ

- คลิกปุ่ม **Upgrade**

![proxmox-repository-6](/images/proxmox/repository-6.png)

- จะมีหน้าต่างป๊อปอัพเด้งขึ้นมา แสดงรายการแพ็คเกจที่ได้รับการอัปเดต จากนั้นกดปุ่ม **Y** แล้วกดปุ่ม **Enter** เพื่อยืนยันการอัปเกรด

![proxmox-repository-7](/images/proxmox/repository-7.png)

- รอจนกว่ากระบวนการอัปเกรดจะเสร็จสมบูรณ์

</Steps>
