# LXC Containers

LXC (Linux Containers) เป็นเทคโนโลยีการจำลองเสมือนที่ช่วยให้คุณสามารถรันหลายระบบปฏิบัติการบนโฮสต์เดียวกันได้อย่างมีประสิทธิภาพ โดยไม่ต้องใช้ทรัพยากรเพิ่มเติมเหมือนกับ Virtual Machines (VMs)

## ข้อดีของ LXC Containers

- **ประสิทธิภาพสูง**: LXC ใช้ทรัพยากรน้อยกว่า VMs เนื่องจากไม่ต้องจำลองฮาร์ดแวร์ทั้งหมด
- **การจัดการง่าย**: สามารถสร้าง ลบ และจัดการคอนเทนเนอร์ได้อย่างรวดเร็ว
- **การแยกทรัพยากร**: LXC ช่วยให้คุณสามารถแยกทรัพยากรของแต่ละคอนเทนเนอร์ได้อย่างชัดเจน เช่น CPU, RAM, และ Storage
- **ความปลอดภัย**: LXC มีการแยกคอนเทนเนอร์แต่ละตัวออกจากกัน ทำให้การโจมตีจากคอนเทนเนอร์หนึ่งไม่ส่งผลกระทบต่อคอนเทนเนอร์อื่น ๆ
