# ขั้นตอนการดาวน์โหลดเทมเพลต

<Steps>

### ล็อกอินเข้าสู่ Proxmox VE Web Interface

เปิดเว็บเบราว์เซอร์และไปที่ URL ของ Proxmox VE ของคุณ

```plaintext
https://<ip_address>:8006
```

### เลือก Node ที่ต้องการเก็บเทมเพลต

ไปที่เมนู **Datacenter** > **Nodes** ที่คุณต้องการเก็บเทมเพลต > **local** > **CT Templates**

![lxc-template-1](/images/proxmox/lxc-template-1.png)

### คลิกที่ปุ่ม **Templates** ที่ด้านบนของหน้า

### เลือกเทมเพลตที่ต้องการดาวน์โหลด 

- เลือกเทมเพลตที่คุณต้องการดาวน์โหลดจากรายการเทมเพลตที่มีอยู่
- คลิกที่ปุ่ม **Download** เพื่อดาวน์โหลดเทมเพลตลงใน Proxmox VE ของคุณ

![lxc-template-2](/images/proxmox/lxc-template-2.png)

- คุณสามารถใช้เทมเพลตที่มีอยู่ใน Proxmox VE หรือดาวน์โหลดเทมเพลตใหม่จากแหล่งอื่น ๆ เช่น [Proxmox Template Repository](http://download.proxmox.com/images/system/)

### รอจนกว่าการดาวน์โหลดจะเสร็จสมบูรณ์

หากดาวน์โหลดสำเร็จ คุณจะเห็นเทมเพลตในรายการเทมเพลตของคุณ

![lxc-template-3](/images/proxmox/lxc-template-3.png)

</Steps>
