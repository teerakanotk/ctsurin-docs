# New record

## ขั้นตอนการเพิ่ม A Record สำหรับ Web server

<Steps>

### เข้าสู่หน้าเว็บจัดการ Technitium DNS

```url copy
http://<ip_address>:5380
```

### เลือก Forward Zone

- ไปที่เมนู **Zones** \> คลิกที่ **Forward Zone ของคุณ** (เช่น ctsurin.local)
- คลิกปุ่ม **Add Record**

![technitium-record-1](/images/technitium/record-1.png)

### เพิ่ม A Record

- **Name:** www
- **Type:** เลือกเป็น A
- **IPv4 Address:** ระบุ IP Address ของ Web Server ที่ต้องการชี้ไป (เช่น 10.11.254.15)
- คลิกปุ่ม **Save**

  ![technitium-record-2](/images/technitium/record-2.png)

</Steps>

## ทดสอบการทำงาน

<Steps>

### เตรียมเครื่องเซิร์ฟเวอร์

เตรียมเครื่องเซิร์ฟเวอร์ที่ต้องการทดสอบ และกำหนดที่อยู่ IP Address เป็นค่าเดียวกันกับ A Record (เช่น 10.11.254.15)

### ติดตั้ง Nginx (Web server)

```bash
sudo apt install -y nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

### ทดลองเข้าเว็บไซต์จากเครื่อง Client

ที่เครื่อง Client เปิดเว็บเบราว์เซอร์และไปที่ `http://www.yourdomain.local`

```url
http://www.ctsurin.local
```

### ผลลัพธ์ที่ควรได้

หากทุกอย่างถูกต้อง คุณควรจะสามารถเข้าถึงหน้าเว็บของ Nginx ได้

![technitium-record-3](/images/technitium/record-3.png)

</Steps>
